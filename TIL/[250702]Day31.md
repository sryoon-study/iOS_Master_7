### 주요 학습내용

자주쓰는 고차함수의 익명변수 정리

| 함수       | 클로저 인자     | 예시                                  | 의미              |
| -------- | ---------- | ----------------------------------- | --------------- |
| `map`    | `$0`       | `items.map { $0.name }`             | 각 요소를 다른 값으로 변환 |
| `filter` | `$0`       | `items.filter { $0.isAvailable }`   | 조건에 맞는 요소만 필터링  |
| `reduce` | `$0`, `$1` | `items.reduce(0) { $0 + $1.price }` | 누적 계산           |
