## 숫자 야구 게임
hamakko branch

### 모델 (Model)
- MenuEnum : 메뉴 열거형
- MsgModel : 메시지 모델
- RecodModel : 기록 관련 모델

### 컨트롤러 (Controller)
- AnswerController : 정답 생성, 판정과 관련 제어
- GameController : 게임 메인 흐름 제어
- InputNumberController : 숫자입력 제어
- msgController : 메시지 출력 제어
- RecordController :  기록 입출력 제어

### 뷰 (View)
- ConsoleView : CLI 출력부
